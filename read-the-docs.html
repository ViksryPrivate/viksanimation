Installation

Get started with VIKS Animation by including it via CDN or installing through npm.

CDN Links

<link href="https://cdn.jsdelivr.net/gh/Vixsry/viks-animation@latest/dist/viks.min.css" rel="stylesheet">
<script src="https://cdn.jsdelivr.net/gh/Vixsry/viks-animation@latest/dist/viks.min.js"></script>

NPM Installation

npm install viks-animation

Basic Usage

Add animations to your elements using data attributes.

Make sure to initialize VIKS after the DOM is loaded.

Example

<div data-viks="fade-up"> This element will fade up </div>

Initialize VIKS

<script> VIKS.init(); </script>

Animation Options

Customize your animations with these available options:

Property Values Description
data-viks-delay number (ms) Delays the animation start time
data-viks-duration number (ms) Sets the animation duration
data-viks-once true/false Whether to repeat the animation
